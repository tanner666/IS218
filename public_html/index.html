Terminal Commands

pwd --- present working directory (gives full path to present directory)
    / forward slash means absolute directory path (from root of drive)
    no forward slash mean relative path
cd <foldername> --- change directory
    cd .. --- makes directory position move back one
    .. is universal signal for reverse
    . is the current directory
    cd ~ --- takes you to home directory
ls --- lists everything inside current directory
    typing a directory after ls and will list inside of directory
mkdir < new folder name> --- creates a new directory
touch <new file name > --- creates an empty file
rm <folder/file name> --- remove folder/file
    rm -r <directory>
    -r means recursive (loops through directory to delete whole directory)
    rm -f
    -f is a force delete (can be used on .git repositories)
mv <oldfilename> <newfilename> --- renames files
mv <file name> <folder name>/
vi <file name> --- opens file editor
    i = insert
    esc = exit insert mode
    shift+: = CL
    w = write
    q = quit
    wq = write and quit

tab key autocompletes names

Git Commands

git init --- creates a new git respository
git status --- current directory and commit status info
git add <filename> --- ("." includes all files)
git commit -m <message describing commit>
git checkout (-b) <branch> (-b makes new branch and switches to it)
git push <remote> <branch> --- pushes changes to github
git merge <branch> --- merges entered branch to current branch

Docker Commands

docker build [options] path/url (of image)
docker run [options] image [command] [args] --- starts a new container
docker ps --- lists containers and their ids
docker stop <container-id>
docker rm <container-id> --- removes container
docker-compose up --- builds, recreates, and starts docker containers
docker-compose down --- stops and removes containers started by up
